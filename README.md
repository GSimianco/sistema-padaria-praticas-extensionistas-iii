# sistema-padaria-praticas-extensionistas-iii


Projeto desenvolvido para a disciplina de Práticas Extensionistas III.

## Descrição do Projeto

Este projeto apresenta a modelagem de um sistema de informação para uma padaria. O sistema tem como objetivo auxiliar no controle de vendas, cadastro de clientes, cadastro de produtos, controle de estoque, registro de pedidos, pagamentos, fornecedores e funcionários.

## Problema

Muitas padarias realizam seus controles de forma manual, o que pode gerar erros no registro de vendas, dificuldade no controle de estoque, perda de informações e demora no atendimento ao cliente.

## Objetivo

Desenvolver a modelagem de um sistema para padaria que permita organizar os processos de venda, atendimento, controle de produtos, estoque e pagamentos.

## Tecnologias Utilizadas

- PHP 8
- MariaDB
- HTML5
- CSS3
- XAMPP
- phpMyAdmin

## Funcionalidades

- Login de usuários
- Dashboard administrativo
- Cadastro de produtos
- Edição de produtos
- Exclusão de produtos
- Consulta e pesquisa de produtos
- Formulário de contato
- Controle de estoque

## Como executar o projeto

1. Instalar o XAMPP.
2. Iniciar Apache e MySQL.
3. Importar o arquivo `padaria_db.sql`.
4. Copiar a pasta `sistema` para `htdocs`.
5. Acessar:

http://localhost/sistema-padaria
